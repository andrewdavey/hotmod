HTML output transformations and modifications for ASP.NET
---------------------------------------------------------

Reference hotmod.dll from your asp.net web application.

Update web.config to look like:

```xml
<?xml version="1.0"?>

<configuration>
  <configSections>
    <section name="hotmod" type="Hotmod.Configuration.HotmodSection, Hotmod"/>
  </configSections>

  <hotmod prettyPrint="InDebugMode" htmlParseError="AppendException">
    <modifiers>
      <!-- add any custom implementations of Hotmod.IModifier here -->
      <add name="foo" type="Demo.TimeStamper, Demo" />
    </modifiers>
  </hotmod>
  
  <system.web>
    <httpModules>
      <add name="hot" type="Hotmod.HtmlTransformer, Hotmod"/>
    </httpModules>
  </system.web>

</configuration>
```

Hotmod will the 'pretty print' the HTML generated by any aspx page and mvc view.
